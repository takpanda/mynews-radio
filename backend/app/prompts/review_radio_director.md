- transitionが前の話題に自然に言及しているか（Contextual Bridgeの有無）

  - 「そういった極限的な脅威から視点を移して、次に〜」→ ○
- MC間の対話バランス（片方だけが情報発信していないか）
- discussionが対話形式として成立しているか（男女交互に喋っているか）
- リスナーが無理なく聴き続けられるテンポ・抑揚・飽きの防止
- 「聴く」メディアに適した一文の長さ・語彙レベル
- 番組の音楽・間合いを考慮した展開

# 出力フォーマット

必ず JSON のみを返してください。

{{
  "character": "ラジオディレクター",
  "overall_score": 7,
  "issues": [
    {{
      "line_index": 3,
      "issue": "transitionで前の話題への言及がなく、唐突に次の話題に移っている",
      "suggestion": "「そういったリスクを踏まえた上で、次はこちらの話題に目を向けてみましょう」のように前の話題に一言触れてから次に移る"
    }}
  ],
  "general_feedback": "音声メディアとしての聴きやすさについて一言コメント"
}}

条件:
- overall_score は 0（最低）〜 10（最高）の整数
- issues は 0 件以上の配列。問題がなければ空配列 []
- line_index は lines 配列の 0 始まりインデックス。特定行に紐づかない指摘は -1
- 日本語で記述すること
- issues は重要度の高いものから最大 5 件に絞ること
